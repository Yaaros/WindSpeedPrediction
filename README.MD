# WindSpeedPrediction

## 项目简介
WindSpeedPrediction 是一个用于风速预测的机器学习项目。该项目采用深度学习模型，结合数据预处理与性能评估模块，实现对风速的高精度预测。

## 目录结构
- **cache/**: 缓存数据目录，存储中间数据以加快后续计算。
- **data/**: 原始数据目录，用于存放原始数据集。
- **dataset/**: 数据预处理后的存储目录，如拆分后的训练集、测试集等。
- **evaluate/**: 模型评估模块，包含模型性能评估的相关代码。
- **log/**: 日志目录，存放运行过程中的日志文件。
- **models/**: 模型定义模块，用于存放深度学习模型的结构代码。
- **plots/**: 图表目录，用于存放训练和评估过程中生成的图表。
- **saved_models/**: 训练后的模型保存目录。
- **train/**: 训练模块，包含训练代码。
- **utils/**: 工具函数模块，包含通用的工具方法。
- **实验报告/**: 实验报告目录，用于存放实验分析文档。
- **废案/**: 废弃内容目录，存放已废弃的代码或方案。
- **app.py**: 应用主入口文件，用于运行整个项目的接口服务。
- **foreign.py**: 外部依赖模块（可能用于与外部系统交互）。
- **main.py**: 项目主程序文件，负责协调各模块的执行。
- **Params.json**: 配置文件，存储项目运行所需的参数。
- **Requirements.txt**: Python依赖文件，列出项目运行所需的第三方库。
- **test.py**: 测试模块，包含单元测试代码。
